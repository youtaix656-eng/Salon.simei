# 指名アップ手帳 — セラピスト向け顧客カルテアプリ

リラクゼーションサロンのセラピストが **指名（リピート指名）を増やす** ための顧客管理アプリです。
お客様一人ひとりを覚えて、最適なタイミングでフォローアップし、「また会いたい」と思ってもらう——
その一連の流れを、**サーバー不要・ブラウザ内で完結** する形でサポートします。
React + Vite で構築し、スマホでの利用を前提にしたPWAです。

## 主な機能

| 機能 | 内容 |
| --- | --- |
| 📇 顧客カルテ | 圧の好み・気になる部位・好きな話題・NG事項・誕生日・カテゴリタグ（常連・VIPなど自由に設定）を記録。次回の接客で「覚えていてくれた」を演出。 |
| 🔍 検索・絞り込み | お客様をカルテ全体（名前・メモ・部位・タグ）からキーワード検索。「7月」で誕生月、「注意」「⚠️」で注意点ありのお客様を検索。タグ・今月誕生日・注意ありのチップで絞り込み。 |
| ✍️ 施術記録 | 日付・メニュー・時間・**指名/フリー**・施術メモ・会話メモをその場で記録。 |
| ⏰ 来店サイクル分析 | お客様ごとの来店間隔（直近5回の平均）から次回来店の目安を予測。**「そろそろ来店時期」「離反リスク」** のお客様をホーム画面に自動リストアップ。 |
| 💌 フォローメッセージ | 来店お礼・来店時期のご案内・お久しぶり・誕生日のテンプレートに、お客様名や経過日数を自動差し込み。生成した文章をコピーして LINE / メールに貼るだけ。テンプレートは自由に編集可能。 |
| 📊 指名率ダッシュボード | 月別の来店数・指名数・指名率のグラフ（直近6ヶ月）、リピート率、平均来店周期、指名の多いお客様ランキング。 |
| 🎯 目標管理 | 月間の指名目標を設定し、ホーム画面で進捗バーを表示。 |
| 🎂 お誕生日アラート | 誕生月のお客様をホーム画面に自動表示。タップしてお祝いメッセージをすぐ作成。 |
| 💰 売上メモ | 施術記録に料金を入力すると、月間売上・指名による売上・平均単価をダッシュボードに集計。 |
| 💡 対処法ノート | 「うつ伏せでおでこが痛くなる方」「腕が痺れやすい方」「肩がほぐれない方」など、体の特徴・お悩み別の対処法を検索・編集できるノート。実践的な初期データ12件入り。 |
| 🤖 AI相談 | 施術・接客・指名アップの悩みを「先輩セラピストAI」に相談できるチャット。カルテからは匿名化した施術情報のみで「このお客様に合う施術プラン」も相談可能。Google Gemini（無料枠あり）または Anthropic Claude の自分のAPIキーを設定して利用。 |
| 💾 バックアップ / 復元 | 全データを1つのJSONファイルに書き出し、別端末で復元可能（APIキーはファイルに含めない）。 |
| 📲 PWA | ホーム画面に追加してアプリのように起動。一度開けばオフラインでも利用可能。 |

## フォローアップ判定のしくみ

お客様ごとの平均来店間隔（直近5回）を基準に、最終来店からの経過日数で状態を判定します。

- **そろそろ来店時期**：間隔の75%を経過（先回りの声かけどき）
- **来店時期を過ぎています**：間隔を超過（フォロー推奨）
- **離反リスク**：間隔の2倍を超過（要ケア）

来店が1回のみのお客様は標準サイクル（30日）で判定します。

## データ保存

- すべてブラウザ内（localStorage）に保存され、**サーバーには送信されません**。
- メッセージもアプリからは送信せず、コピーして普段お使いの LINE / メール等に貼り付ける運用です。
- 端末間で持ち運ぶ場合は、設定画面の「バックアップと復元」でJSONファイルをやり取りします。
- 設定画面の「デモデータを読み込む」で、動作をすぐに確認できます。
- AI相談を使う場合のみ、質問文がブラウザから選択したAIプロバイダ（Google / Anthropic）へ直接送信されます。APIキーは端末内にのみ保存され、バックアップファイルには含まれません。お客様の実名など個人情報は質問に書かない運用を推奨します。

## 開発・ビルド

```bash
npm install      # 依存関係のインストール
npm run dev      # 開発サーバー（http://localhost:5173）
npm run build    # 本番ビルド（dist/ に出力）
npm run preview  # ビルド結果のプレビュー
npm test         # ユニットテスト（cycle / stats / messages / storage）
```

`dist/` は静的ファイルのみで動作するため、GitHub Pages など任意の静的ホスティングに配置できます
（`vite.config.js` で相対パス `base: './'` を設定済み）。

## GitHub Pages への自動デプロイ

`.github/workflows/deploy.yml` を用意しています。`main` ブランチへ push すると、
テスト → ビルド → デプロイが自動実行されます。初回のみ次の設定が必要です。

1. GitHub リポジトリの **Settings → Pages** を開く
2. **Source** を **「GitHub Actions」** に設定する

以後、`main` への push ごとに `https://<ユーザー名>.github.io/<リポジトリ名>/` へ公開されます。

## テスト

純粋ロジック（来店サイクル分析・統計・テンプレート差し込み・バックアップの読み書き）に
Node 標準のテストランナーでユニットテストを用意しています（`npm test`）。
CI（GitHub Actions）でもデプロイ前に実行されます。

## 技術構成

- **React 18** + **Vite**（外部サーバー・API課金なし）
- 永続化は localStorage
- グラフは依存ライブラリなしの自前SVG実装

## ディレクトリ構成

```
src/
  components/   各画面（Home, Clients, ClientDetail, VisitForm, Dashboard, Settings）
  lib/          ロジック（cycle, stats, messages, storage, useStore）
  data/         デモデータ
  styles/       スタイル
test/           ユニットテスト
```

## 注意事項

- スマホ利用がメイン。レスポンシブ対応済み。
- お客様の個人情報を扱うアプリです。端末のロックを設定のうえ、バックアップファイルの管理にもご注意ください。
